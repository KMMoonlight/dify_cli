# dify_cli

1. 安装对应的requests库, 使用 `pip3 install requests`
2. 下载文件,放置在Home目录下
3. 使用`chmod +x ./dify.py` 授予运行权限
4. 编辑bash_profile.sh 或者 .zshrc 文件, 添加 `alias dify='~/dify.py'` 和 `alias difyc='~/dify.py completion'` 这里的别名可以自行选取
5. 输入 :q, :exit, :quit内容来退出 


> dify 适合对话式
> difyc 适合文本生成 翻译等
